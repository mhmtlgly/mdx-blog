## Convert a number to Euro

```js
export const amountToEuro = amount =>
  new Intl.NumberFormat("de-DE", {
    style: "currency",
    currency: "EUR",
    minimumFractionDigits: 2,
    // currencyDisplay: "symbol"
  }).format(amount)

amountToEuro(20)

```
