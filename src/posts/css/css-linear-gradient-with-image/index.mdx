## CSS linear-gradient with Image

```css
div {
  background: linear-gradient(
      0deg,
      rgba(255, 0, 150, 0.3),
      rgba(255, 0, 150, 0.3)
    ), url(http://lorempixel.com/800/600/nature/2);
  /* url(${imgSrcVar}) center center / contain no-repeat`} */
}
```
